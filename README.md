# 🎤 Kalam.me

**Application d'apprentissage de l'arabe avec IA Gemini**

## 🚀 Fonctionnalités

- 🎓 3 Leçons structurées (Salutations, Se présenter, Basmala)
- 📚 Vocabulaire arabe littéraire avec transcription et traduction
- 🤖 Professeur IA (Gemini 2.0 Flash) qui répond en arabe et français
- 🎤 Reconnaissance vocale (Web Speech API)
- 🔊 Text-to-Speech pour écouter la prononciation
- 📱 Design responsive mobile-first

## 🎯 Comment utiliser

1. Ouvrez l'app: https://medinimmersion.github.io/Kalam.me-by-M-dinImmersion-
2. Sélectionnez une leçon
3. Écoutez le vocabulaire
4. Cliquez sur le bouton 🎤 pour parler
5. Recevez du feedback du professeur IA

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript vanilla
- Web Speech API
- Gemini API

## 📝 Leçons

### Leçon 1: Salutations
- السلام عليكم (Assalam alaykoum)
- صباح الخير (Sabah al-khair)
- مساء الخير (Masa al-khair)
- وداعا (Wada'an)

### Leçon 2: Se présenter
- من أين أنت؟ (Man ayna anta?)
- أنا من (Ana min)
- اسمي (Ismi)

### Leçon 3: Basmala
- بسم الله الرحمن الرحيم (Bismillah ar-rahman ar-rahim)
- الحمد لله (Al-hamdu lillah)
