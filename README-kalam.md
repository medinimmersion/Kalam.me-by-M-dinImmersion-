# 🎤 Kalam.me - Apprenez l'ararabe en parlant!

**Kalam.me** est une application web interactive pour apprendre l'arabe littéraire avec un professeur IA (Gemini Live).

## ✨ Fonctionnalités

- 🎤 **Prononciation vocale** - Parlez en arabe et obtenez du feedback
- 🎓 **Leçons structurées** - Salutations, se présenter, lire le Coran
- 📚 **Vocabulaire** - Arabe littéraire + transcription + traduction
- 🤖 **Professeur IA** - Gemini Live répond avec corrections et encouragements
- 📱 **Mobile-first** - Optimisé pour tous les appareils

## 🚀 Accès

Ouvrez simplement:
```
https://medinimmersion.github.io/Kalam.me-by-M-dinImmersion-
```

## 🎯 Comment utiliser

1. **Choisissez une leçon** (Salutations, Se présenter, Basmala)
2. **Voyez le vocabulaire** avec transcription et traduction
3. **Commencez l'appel vocal** 🎤
4. **Parlez en arabe!** Le micro enregistre votre prononciation
5. **Recevez du feedback** de votre professeur Abou Adam

## 🛠️ Technologies

- **HTML5** - Structure et sémantique
- **CSS3** - Design responsive et ludique
- **JavaScript vanilla** - Sans dépendances
- **Web Speech API** - Reconnaissance vocale et synthèse vocale
- **Gemini API** - IA pour réponses du professeur
